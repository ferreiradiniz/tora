## Instalação

Instalar dependência e estartar o server

```sh
cd server
npm i
npm start
```
Instalar dependência e estartar o cliente
```sh
cd client
npm i
npm run dev
```
Teste unitário servidor
```sh
cd server
npx jest
```

## OBS
